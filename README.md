# Blockchain F1 Ticketing System

This is a decentralized application (dApp) for the F1 Ticketing System, built with Solidity, MetaMask, and Ethereum smart contracts.

## Features
- Smart contracts for F1 ticket transactions
- Secure and transparent ticketing system

## Technologies
- Solidity
- MetaMask
- Ethereum Blockchain

## Installation
1. Clone this repository
2. Install dependencies
3. Deploy the smart contracts on the Sepolia test network
4. Install Simple Web Server
5. Run the codes via Simple Web Server
